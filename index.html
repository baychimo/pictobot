Pictobot¶

Pictobot is a Telegram Bot, that is, a conversational program. It is designed to help you communicate while you are traveling abroad.

You are reading the docs generated from the docstrings in the pictobot project. The source code can be viewed on Github here: https://github.com/baychimo/pictobot

Over there you’ll find info on how to get started, install the environment and run your own pictobot! Or simply fork the code and take it somewhere else! Or you’re not lost and you can keep on reading the docs here:
